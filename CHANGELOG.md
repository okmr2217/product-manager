# Changelog

## 1.1.0 - 2026-04-21

- **type**: MINOR
- **title**: UIデザインの大幅刷新と操作性改善

### 追加

- favicon と Web App Manifest（PWA 対応）を追加しました

### 改善

- アプリ名を「Launchpad」に変更しました
- デザインテーマをインディゴ×ダークサイドバー（Control Room）に刷新しました
- ダッシュボードにグリッド / テーブル表示切り替えを追加し、デフォルトを最新リリース順に変更しました
- プロダクト詳細ページを刷新しました（概要サマリー・基本情報タブ・ピル型セグメント UI・タブ件数バッジ）
- 画像管理 UI を全面刷新しました（デバイスタイプ分類・常時 D&D 対応・プレビュー拡大・2カラムモーダル）
- リリース日をステータス履歴から自動取得するよう変更し、手動入力フィールドを廃止しました
- フォームの使いやすさを改善しました（日時入力のデフォルトを当日に、下書きのデフォルトをオフに変更）
- セッション有効期限を 90 日に延長しました

## 1.0.0 - 2026-03-27

- **type**: MAJOR
- **title**: 初回リリース — プロダクト管理アプリ

### 追加

- メール + パスワード認証（Better Auth）
- プロダクトの作成・編集・削除（名前・スラッグ・説明・技術スタック・URL・公開設定など）
- ダッシュボードのフィルタ（ステータス・カテゴリ）とソート機能
- ステータス管理と変更履歴の記録（変更日時の指定・履歴の編集・削除に対応）
- 画像のアップロード・サムネイル指定・並び替え
- リリースノートの作成・編集・削除（マークダウン全文レンダリング対応）
- DevTask（開発タスク）の CRUD・フィルタ・完了トグル
- サイドバーでプロダクトを切り替えたときに現在のタブを引き継ぐ機能
- モバイル向けサイドバー（ハンバーガーメニュー）
